## \f1\u10024?\f2\lang1033  \f3 Highlights | \f0\'ba\'cb\'d0\'c4\'c1\'c1\'b5\'e3\par
\par
### 1. **Formulaic Prompt Construction | \'b9\'ab\'ca\'bd\'bb\'af\'d6\'f7\'cc\'e5\'b9\'b9\'bd\'a8**\par
*   **Logical Matrix**: Prompts are broken down into **Noun, Action, Location, Time, Environment, and Result**.\par
*   **Atomic Building Blocks**: Users can pick "Lego-like" words (e.g., *Old man, Smiling, Victorian study*) instead of rigid long sentences.\par
*   **\'c2\'df\'bc\'ad\'be\'d8\'d5\'f3**\'a3\'ba\'bd\'ab\'cc\'e1\'ca\'be\'b4\'ca\'b2\'f0\'bd\'e2\'ce\'aa**\'c3\'fb\'b4\'ca\'a1\'a2\'b6\'af\'d7\'f7\'a1\'a2\'b5\'d8\'b5\'e3\'a1\'a2\'ca\'b1\'bc\'e4\'a1\'a2\'bb\'b7\'be\'b3\'a1\'a2\'bd\'e1\'b9\'fb**\'c1\'f9\'b4\'f3\'ce\'ac\'b6\'c8\'a1\'a3\par
*   **\'d4\'ad\'d7\'d3\'bb\'af\'bb\'fd\'c4\'be**\'a3\'ba\'d3\'c3\'bb\'a7\'bf\'c9\'d2\'d4\'cf\'f1\'c6\'b4\'c0\'d6\'b8\'df\'d2\'bb\'d1\'f9\'d7\'d4\'d3\'c9\'d7\'e9\'ba\'cf\'b4\'ca\'bb\'e3\'a3\'a8\'c8\'e7\'a3\'ba*\'c0\'cf\'c8\'cb\'a1\'a2\'ce\'a2\'d0\'a6\'a1\'a2\'ce\'ac\'b6\'e0\'c0\'fb\'d1\'c7\'ca\'e9\'b7\'bf*\'a3\'a9\'a3\'ac\'b6\'f8\'b7\'c7\'cb\'c0\'b0\'e5\'b5\'c4\'b3\'a4\'be\'e4\'a1\'a3\par
\par
### 2. **Data-Logic Decoupling | \'ca\'fd\'be\'dd\'d3\'eb\'c2\'df\'bc\'ad\'b3\'b9\'b5\'d7\'bd\'e2\'f1\'ee**\par
*   **JSON Driven**: All categories and vocabularies are stored in an external `lexicon.json`.\par
*   **Zero-Code Customization**: Users can expand the library by simply editing the JSON file without touching the Python core.\par
*   **\'ca\'fd\'be\'dd\'c7\'fd\'b6\'af**\'a3\'ba\'cb\'f9\'d3\'d0\'b7\'d6\'c0\'e0\'d3\'eb\'b4\'ca\'bf\'e2\'be\'f9\'b4\'e6\'b4\'a2\'d4\'da\'cd\'e2\'b2\'bf `lexicon.json` \'d6\'d0\'a1\'a3\par
*   **\'c1\'e3\'b4\'fa\'c2\'eb\'d7\'d4\'b6\'a8\'d2\'e5**\'a3\'ba\'d3\'c3\'bb\'a7\'d6\'bb\'d0\'e8\'b1\'e0\'bc\'ad JSON \'ce\'c4\'bc\'fe\'bc\'b4\'bf\'c9\'c0\'a9\'c8\'dd\'b4\'ca\'bf\'e2\'a3\'ac\'ce\'de\'d0\'e8\'d0\'de\'b8\'c4 Python \'d4\'b4\'b4\'fa\'c2\'eb\'a1\'a3\par
\par
### 3. **High-Fidelity Offline TTS | \'b5\'e7\'d3\'b0\'bc\'b6\'c0\'eb\'cf\'df\'d3\'ef\'d2\'f4**\par
*   **ChatTTS Integration**: Powered by the **ChatTTS** engine, providing natural, human-like voice synthesis with "breathing" and "pauses".\par
*   **No API Needed**: 100% local operation with zero Token cost and full privacy protection.\par

*   **Python 3.14+** (Optimized for cutting-edge environments)\par

```bash\par
